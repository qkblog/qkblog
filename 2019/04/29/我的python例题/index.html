题目1：实现一个数字游戏，1-100之间随机猜一个数字，猜错继续，猜对结束
代码：shuzi.py
import random
print(‘*这是钱锴写的数字小游戏**‘)
print(‘***欢迎体验**‘)
a=random.randint(1,100)
count=0
while True:
b=input(‘请你输入一个1到100之间的整数：’)
if b.isdigit():
count += 1
b=int(b)
if b==a:
print(‘恭喜你猜对了，你猜的次数是’,count)
c=input(‘尊敬的玩家，您是否要再次玩这个游戏(y or n):\n’)
if c==’y’:
count=0
a=random.randint(1,100)
continue
elif c==’n’:
print(‘#############游戏结束了，欢迎下次在玩###########’)
break
else:
print(‘输入的格式不对，游戏自动退出’)
break
elif b<a:
print(‘你猜的数小了\n’)
elif b>a:
print(‘你猜的数大了\n’)
else:
print(‘输入的值不符合标准！’)
continue

题目2：任意输入2个整数，会输出这2个整数之间的素数和偶数
代码：sushu.py
s=[]
sum=input(‘请输入任意2个整数：’)
for i in sum.split():
i=int(i)
s.append(i)

s=sorted(s)
q=[i for i in range(int(s[0]),int(s[1])+1) if i % 2 == 0]
k=[i for i in range(int(s[0]),int(s[1])+1) if i % 2 != 0]

print(‘整数’,s[0],’和整数’,s[1],’之间的偶数有：’,q)
print(‘整数’,s[0],’和整数’,s[1],’之间的素数有：’,k)
